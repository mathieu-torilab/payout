Flow Configuration

Configure the number of approval steps for each workflow and assign a permission group to each step.

Each step must be assigned a different permission group. The uploader or request receiver cannot be in the approving group for their own submission.
Changes apply to new requests only. Earnings files and payout requests already in progress will continue under the configuration that was active when they were created. Changing the approver group for a step takes effect immediately for pending steps not yet acted upon.
